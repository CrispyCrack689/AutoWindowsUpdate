# AutoWindowsUpdate

**生成AIによって作成**

**AutoWindowsUpdate** は、Windows Update を自動的に実行するための C# コンソールアプリケーションです。
未インストールの更新プログラムを自動的に検索し、ダウンロードおよびインストールを行います。

## 概要

このツールを実行すると、以下のプロセスが自動的に行われます：

1.  **検索**: システムに適用可能な未インストールの更新プログラムを検索します。
2.  **ダウンロード**: 見つかった更新プログラムをすべてダウンロードします（進捗表示あり）。
3.  **インストール**: ダウンロードされた更新プログラムをインストールします（進捗表示あり）。
4.  **結果報告**: インストール結果と、再起動が必要かどうかを表示します。

## 機能

- **完全自動化**: ユーザーが操作することなく、検索からインストールまでを一貫して行います。
- **進捗バー表示**: ダウンロードおよびインストールの進行状況をコンソール上で可視化します。
- **多言語対応**: システムの言語設定（ロケール）を自動検出し、日本語環境では日本語で、それ以外では英語でメッセージを表示します。
- **管理者権限チェック**: アプリケーションが適切な権限で実行されているかを起動時にチェックします。

## 動作環境

- OS: Windows 10 / 11 / Server (Windows Update Agent API が利用可能な環境)
- ランタイム: .NET 10.0 (またはビルド時のターゲットフレームワーク)
- **権限**: 実行には「管理者権限」が必要です。

## 使い方

1.  アプリケーションをビルドします。
    ```powershell
    dotnet build
    ```

2.  生成された実行ファイル（`.exe`）を、**管理者として実行**します。
    - コマンドプロンプトまたは PowerShell を管理者として開き、実行ファイルを実行してください。

    ```powershell
    cd bin\Debug\net10.0-windows
    .\AutoWindowsUpdate.exe
    ```

3.  処理が完了するまで待ちます。
    - インストール完了後、再起動が必要な場合はメッセージが表示されます。手動で再起動を行ってください。

## 注意事項

- 本ツールは Windows Update Agent API (COM) を使用しています。
- システムの重要な更新を行うため、実行前に作業中のデータを保存することを推奨します。
